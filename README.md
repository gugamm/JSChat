# JSChat

JSChat is a chat written in javascript using:

  - [ReactJs] [Rjs]
  - [Socket IO] [SIO]
  - [Flux] [Flx]

### Version
0.0.1

### Installation

Clone repo using:

```sh
$ git clone
```

Install dependencies:

```sh
$ npm install
```

### Scripts

Build your bundle into out/bundle.js
```sh
$ npm run build
```

Watch for changes and automatically update your bundle.js
```sh
$ npm run watch
```
